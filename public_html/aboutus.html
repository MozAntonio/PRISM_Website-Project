Cos'è Prism Game Reviews

Prism Game Reviews è un sito italiano dedito alla recensione di videogiochi.

I videogiochi recensiti spaziano su qualsiasi genere e piattaforma di gioco, senza alcuna distinzione.

La missione di Prism Game Reviews

Noi di Prism Game Reviews vogliamo permettere a chiunque di poter leggere le nostre recensioni sui videogiochi d'interesse. Proponendo in modo critico le nostre impressioni ed incitando i nostri utenti ad esprimere attivamente i loro pensieri, critiche ed approvazioni mediante i commenti alle recensioni.

Il nostro giudizio tenta sempre di essere equo, nonostante le inevitabili preferenze personali, allo scopo di guidare gli utenti ad un eventuale acquisto più accurato ed informato nel crescente mare di videogiochi disponibili.

Il nostro obiettivo è rivolgersi principalmente al pubblico italiano, ma recensiamo qualsiasi tipologia di videogiochi (perlopiù internazionali).

L'iscrizione al sito è totalmente gratuita e senza scopo di lucro o finalità commerciale.

Il progetto

Il sito Prism Game Reviews è stato interamente realizzato come progetto didattico per l'insegnamento di Tecnologie Web dell'anno accademico 2016-2017 erogato al terzo anno del corso di laurea in informatica all'università di Padova.

Obiettivi formativi

Il nostro obiettivo è stato quello di applicare la maggior parte delle nozioni e degli insegnamenti appresi durante le lezioni frontali e nei laboratori didattici, esplorando in modo più approfondito tutte le tecnologie studiate con un occhio di riguardo per le norme riguardanti l'accessibilità e la struttura (anche semantica) delle pagine web.

Le tecnologie principali da noi utilizzate sono:

  • HTML5 (nel rispetto dello standard XHTML Strict)
  • CSS 2.1 ed alcune specifiche/moduli di CSS 3
  • PHP 7.1
  • MySQL 5.7
  • JavaScript
